# Formemo

ForMemo è un'app iOS progettata per aiutarti a organizzare attività e promemoria in modo semplice, intuitivo ed efficace.

## ✨ Funzionalità

- Creazione rapida di attività
- Promemoria intelligenti
- Supporto agli allegati
- Sincronizzazione tra dispositivi
- Interfaccia personalizzabile
- Integrazione con Siri
- Supporto multilingua (Italiano, Inglese, Francese, Tedesco, Spagnolo)

## 📱 Descrizione

ForMemo ti aiuta a catturare, organizzare e ricordare ciò che conta davvero.  
Crea attività in modo naturale, arricchiscile con allegati e ricevi promemoria puntuali, al momento giusto.
